# bat-video-compressor
.bat script for quick hardware-accelerated video compression using ffmpeg

# How to use
Unpack archive from [releases](https://github.com/FurryTr4sh/bat-video-compressor/releases) to any folder you prefer\
Drag and drop video (or even multiple) on .bat file with preferred resolution and framerate. If you need to preserve original resolution and framerate, use "OrigRes"
